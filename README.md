# 🎮 Tic-Tac-Toe GUI — Python & Tkinter

Juego de Tic-Tac-Toe con interfaz gráfica desarrollado como práctica de Git/GitHub.

## 📦 Instalación

1. Clona el repositorio:
```bash
   git clone https://github.com/20241d020-hue/tic-tac-toe-gui.git
   cd tic-tac-toe-gui
```
2. Asegúrate de tener Python 3.x instalado (Tkinter viene incluido).
3. Ejecuta:
```bash
   python main.py
```

## 🕹️ Cómo jugar

- Dos jugadores se turnan: X y O
- Haz clic en cualquier celda para marcar
- Gana quien complete una fila, columna o diagonal
- Usa el botón 🔄 Reiniciar para nueva partida

## 🌿 Ramas utilizadas

| Rama | Descripción |
|------|-------------|
| `main` | Rama principal con versión estable |
| `feature-logic` | Lógica de detección de ganador |
| `feature-gui-styling` | Estilos visuales y tema oscuro |
| `feature-reset` | Botón de reinicio del tablero |
